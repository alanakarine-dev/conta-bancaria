# 💰 Sistema de Conta Bancária

##  Descrição

Aplicação desenvolvida em Java que simula um sistema bancário simples, utilizando os conceitos de Programação Orientada a Objetos (POO). O sistema permite realizar operações básicas como saque, depósito e consulta de informações da conta.

---

## ⚙️ Funcionalidades

*  Realizar depósito
*  Realizar saque
*  Consultar saldo da conta
*  Verificar status da conta (aberta ou fechada)
*  Identificar o tipo de conta (corrente ou poupança)

---

## 🛠️ Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

---

## 📚 Conceitos aplicados

* Classes e objetos
* Encapsulamento
* Métodos
* Atributos
* Abstração
